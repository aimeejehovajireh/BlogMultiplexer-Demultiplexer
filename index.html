Apa itu MUX & DEMUX?

Dalam elektronika digital, Multiplexer dan Demultiplexer bekerja secara berlawanan untuk mengelola aliran data:

  • Multiplexer (MUX): Multiplexer, atau sering disingkat MUX, adalah perangkat elektronika digital yang berfungsi sebagai sakelar pemilih data otomatis. Dalam hierarki logika digital, MUX dikategorikan sebagai perangkat n-to-1, yang berarti ia memiliki banyak jalur input tetapi hanya memiliki satu jalur output tunggal. Fungsi utamanya adalah menerima berbagai sinyal digital dari sumber yang berbeda dan memilih salah satu dari sinyal tersebut untuk diteruskan ke saluran keluar pada waktu tertentu. Secara garis besarnya, Multiplexer berperan sebagai pemilih (Selector). Ia menggabungkan banyak jalur input menjadi satu output.

  • Demultiplexer (DEMUX): Demultiplexer adalah kebalikan logis dari Multiplexer. Jika MUX mengumpulkan data, maka DEMUX berfungsi sebagai penyalur atau distributor data digital. Perangkat ini dikategorikan sebagai 1-to-n, yang berarti ia menerima data dari satu jalur input utama dan menyebarkannya ke salah satu dari banyak jalur output yang tersedia. DEMUX memastikan bahwa data yang dikirim melalui satu jalur transmisi dapat sampai ke tujuan atau perangkat penerima yang spesifik dan tepat. Secara garis besarnya, Demultiplexer berperan sebagai penyalur (Distributor). Ia memecah satu input kembali menjadi banyak output.

Prinsip Kerja & Tabel Kebenaran

1. Multiplexer (MUX 4-ke-1)

Multiplexer sangat krusial dalam sistem pemrosesan data dan komunikasi karena memungkinkan pengiriman informasi dari berbagai sumber melalui satu kabel atau frekuensi saja, yang secara teknis disebut sebagai teknik multiplexing. Memilih input (D0 - D3) berdasarkan nilai pada Select Lines ($S_1, S_0$).

Select S1 Select S0 Output (Y)
0 0 Data dari Input D0
0 1 Data dari Input D1
1 0 Data dari Input D2
1 1 Data dari Input D3

2. Demultiplexer (DEMUX 1-ke-4)

Menyalurkan satu input ke salah satu output (Y0 - Y3).

Select S1 Select S0 Jalur Output Aktif
0 0 Disalurkan ke Y0
0 1 Disalurkan ke Y1
1 0 Disalurkan ke Y2
1 1 Disalurkan ke Y3
